# Git Commands for Kenal Admin

## Initialize and Push to GitHub

```bash
# 1. Initialize git (if not already)
git init

# 2. Add all files
git add .

# 3. Commit with message
git commit -m "feat: Complete MVP of Kenal Admin Dashboard

- ✅ Authentication system (admin only)
- ✅ Interactive dashboard with charts
- ✅ User management with filters
- ✅ Dark/Light theme support
- ✅ Mobile responsive design
- ✅ Connected to production Supabase

Ready for Cursor AI handover."

# 4. Add your GitHub remote (replace with your repo URL)
git remote add origin https://github.com/YOUR_USERNAME/kenal-admin.git

# 5. Push to main branch
git push -u origin main
```

## After Push
1. Open project in Cursor
2. Read CURSOR-HANDOVER.md
3. Run `./start-dev.sh` to start development

## Files to Check
- ✅ .gitignore (configured)
- ✅ README.md (created)
- ✅ CURSOR-HANDOVER.md (created)
- ✅ .env.local (in .gitignore, won't be pushed)
